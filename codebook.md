# Codebook

Study: Business Intelligence Utilisation and Decision-Making Quality in Kent SMEs
Sample: n = 186 respondents from small and medium-sized enterprises in Kent, United Kingdom
Collection period: 2025
IRB approval: 2025-006

All Likert items use a 5-point agreement scale: 1 = Strongly disagree, 2 = Disagree, 3 = Neutral, 4 = Agree, 5 = Strongly agree. Exact item wording as administered is given in `survey_instrument.pdf`.

There are no missing values in the dataset.

---

## File 1: data_anonymised.csv

Raw survey responses with the collection timestamp removed for anonymisation. 19 variables, 186 cases.

### Demographics

| Variable | Description | Coding |
|---|---|---|
| Job_Position | Respondent's role in the firm | 1 = CEO, 2 = Senior Manager, 3 = Middle Manager, 4 = Operational Manager, 5 = Other |
| Industry | Sector of the firm | 1 = Manufacturing, 2 = Retail, 3 = Services, 4 = Technology, 5 = Other |

### Strategic BI utilisation (3 items)

| Variable | Description |
|---|---|
| Strategic_BI_Availability | Availability of BI for strategic decisions |
| Strategic_BI_Alternatives | Use of BI to evaluate strategic alternatives |
| Strategic_BI_Alignment | BI support for aligning decisions with long-term goals |

### Tactical BI utilisation (3 items)

| Variable | Description |
|---|---|
| Tactical_BI_Availability | Availability of BI for tactical decisions |
| Tactical_BI_Resource_Alloc | Use of BI in resource allocation |
| Tactical_BI_Data_Decisions | Use of BI data in mid-level decisions |

### Operational BI utilisation (3 items)

| Variable | Description |
|---|---|
| Operational_BI_Availability | Availability of BI for day-to-day operations |
| Operational_BI_Agility | BI support for operational agility and responsiveness |
| Operational_BI_Quality | Quality of BI information at the operational level |

### Perceived decision-making quality, DMQ (4 items; dependent variable)

| Variable | Description |
|---|---|
| BI_Improves_Decisions | BI improves the quality of decisions |
| BI_Speed_Accuracy | BI improves the speed and accuracy of decisions |
| BI_Data_Inform | Decisions are informed by BI data |
| BI_Alternative_Solutions | BI helps identify alternative solutions |

### BI adoption challenges (4 items; descriptive only, not in the regression models)

| Variable | Description |
|---|---|
| BI_Cost_Barrier | Cost as a barrier to BI adoption |
| BI_Training_Limit | Lack of training limits BI use |
| BI_Flexibility_Issue | Limited flexibility of BI tools |
| BI_Data_Integration_Issue | Difficulty integrating data sources |

## File 2: data_processed.csv

The same 186 cases after SPSS processing. Contains the 19 substantive variables above (timestamp excluded) plus variables generated by `SPSS_syntax_verification.sps`:

| Variable | Description |
|---|---|
| Strategic_Factor, Tactical_Factor, Operational_Factor, DMQ_Factor, Challenges_Factor | Factor scores saved from per-construct Principal Axis Factoring (regression method) |
| Z[item name] | Standardised (z) scores of the 13 BI and DMQ items |
| filter_$ | SPSS case filter flag (0/1) |
| Strategic_Mean, Tactical_Mean, Operational_Mean | Composite means of the three items per BI construct, used in the robustness regression |
| Decision_Making_Quality | Composite mean of the four DMQ items |
| MAH_1 | Mahalanobis distance from the regression diagnostics |

All derived variables in this file are reproducible by running the syntax file against `data_anonymised.csv`.

## Anonymisation note

Submission timestamps were removed prior to deposit. No names, contact details, IP addresses, firm names, or free-text responses were collected in the deposited variables.
